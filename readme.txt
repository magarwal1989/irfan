Requirement:

1. Download maven on your system
2. add MAVEN_HOME
3. add maven bin path to path variable in system
4. open cmd and check "mvn -version"
5. maven version should display

steps to run:

1. go to File-> Import-> Maven-> Existing Maven Project
2. open the directory in which pom.xml is there
3. open the command prompt-> write-> mvn clean
4. mvn install
5. mvn test

To view report

1. go to project directory
2. open target folder
3. Open surefire-reports folder
4. open index.html file
